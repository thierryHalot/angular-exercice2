My solution :

{{username}}

Explication :

In this exercise, I first created a username property which I linked with my input via the [(ngModel)] directive.
For this to work, I had to import FormsModule into the app.modules.ts file.
Then I created a button with a [disabled] attribute that I linked with my username property, when username is empty, it will return false and true if text is entered, which allows the activation and deactivation of the button.
Then when a click event is performed on my button, I simply execute a function that empties my username property.
